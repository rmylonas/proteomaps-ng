Some documentation

MixMHCp is a website to identify clusters within large datasets of short peptides.
The different clusters are mathematically modelled with Position Weight Matrices (PWMs) that can be visualized with sequence logos and are often referred to as "motifs".

The algorithm automatically learns the entries of these PWMs and their respective weight.
The learning is based on mixture models and maximum likelihood (EM algorithm).

Input

As input, you should give a list of peptides, all of the same length and consisting only of natural amino acids.
The list of peptides can be in standard text format with each peptide on a different line, or in fasta format.
Be careful not to load files with metadata, such as Excel spreadsheets.
"Max nr of motifs" corresponds to the maximal number of motifs that should be explored (Maximal value is 20).

Output

Graphical output

The different logos representing each cluster are shown for different numbers of clusters (up to "Max nr of motifs"), as well as the number of peptides assigned to them.

Downloadable output

  • The responsibility/ folder contains the assignment of each peptide to each cluster for the different number of clusters.
    These values correspond to the contribution of each peptide to each cluster or motif.
    Peptide assignment to each cluster can be done by considering the max of the responsibility values.
  • The Multiple_PWMs/ folder contains the PWMs that optimally model the input data.
    Of note, these are not meant to be used as predictors, since they do not include any renormalization for background frequencies
    and the prior used in the mixture model may not be optimal for training predictors.
  • The logos/ folder contains the logos (.png) representing each cluster.
    Logos have been generate with the LoLa software.
    The logos.html file allows you to quickly visualize all the logos.
  • The data/ folder contains the input data.
  • The KLD/ folder contains the KLD scores for each number of motifs, as well as the predicted optimal number of motifs.
    However, it is in general advisable to manually go through the motifs to check the optimal number of motifs, since this number may depend on the input data and the kind of application that you want.

Other information

MixMHCp algorithm is developed by the Computational Cancer Biology Lab at the Ludwig Center for Cancer Research at UNIL headed by Prof. David Gfeller, in collaboration with the Vital-IT team.

In the past, the method has been successfully applied to phage display data to identify multiple specificity patterns [Gfeller et al, MSB 2011; Kim et al. NAR 2012] and HLA peptidomics data to deconvolute mixtures of HLA ligands coming from up to 6 alleles [Bassani-Sternberg and Gfeller, JI 2016; Bassani-Sternberg et al. bioRxiv 2017].

If you use this website in a publication, please cite [Bassani-Sternberg M and Gfeller D, J. Immunol. 197(6):2492 (2016)]
