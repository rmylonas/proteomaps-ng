Some documentation

MixMHCp is a website to identify clusters within large datasets of short peptides. The different clusters are modelled with distinct Position Weight Matrices (PWMs) that can be visualized with sequence logos.
The algorithm automatically learns the entries of these PWMs and their respective weight. The learning is based on mixture models and maximum likelihood (EM algorithm), which scales linearly with the size of the dataset.

Input

As input, you should give a list of peptides, all of the same length and consisting only in natural amino acids. The list of peptides can be in standard text format with each peptide on a different line, or in fasta format. Max nr of motifs corresponds to the maximal number of motifs that should be explored (Maximal value is 20).

Output

Graphical output

The different logos representing each cluster are shown for different numbers of clusters (up to "Max nr of motifs"), as well as the number of peptides assigned to them.

Downloadable output

The responsibility/ folder contains the assignment of each peptide to each cluster. These files correspond to the contribution of each peptide to each motif. Peptide assignment to each cluster can be done by considering the max of the responsibility values.
The Multiple_PWM/ folder contains the PWMs that optimally model the input data. Of note, these are not meant to be used for predictions, since they do not include any renormalization fot background frequencies, or
The logos/ folder contains the logos representing each cluster. The logos.html file allows you to quickly visualize the logos for the different number of clusters.
The data/ folder contains the input data.
The KLD/ folder contains the KLD score for each number of motifs, as well as the predicted optimal number of motifs. However, it is in general advisable to manually go through the motifs to check the optimal number of motifs, since this number may depend on the input data and the kind of application that you want.

Other information

MixMHCp is developed by the Computational Cancer Biology Lab at the Ludwig Center for Cancer Research at UNIL headed by Prof. David Gfeller, in collaboration with the Vital-IT team.
In the past, the method has been successfully applied to phage display data to identify multiple specificiy patterns [Gfeller et al, MSB 2011; Kim et al. NAR 2012] and HLA peptidomics data to deconvolute mixtures of HLA ligands coming from up to 6 alleles [Bassani-Sternberg and Gfeller, JI 2016; Bassani-Sternberg et al. bioRxiv 2017].
